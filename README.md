# 📊 Client 360 Investment Analysis Dashboard

## 📌 Project Overview

This project provides a **360-degree view of client investments**, enabling analysis of portfolio performance, client behavior, and investment trends.

The dashboard helps stakeholders:

* Track **total investment & returns**
* Identify **high-value clients**
* Analyze **portfolio growth trends**
* Evaluate **risk distribution across assets**

---

## 🎯 Business Problem

Financial institutions need a centralized system to:

* Monitor client portfolios
* Identify profit/loss patterns
* Segment clients for better decision-making

This project solves that by building an **interactive dashboard** with deep analytical insights.

---

## 🛠️ Tools & Technologies

* Power BI (Dashboard & Visualization)
* PostgreSQL (Data Analysis)
* Python (Data Cleaning - optional)
* Excel (Dataset)

## 📊 Key Metrics

* Total Clients: 1000
* Total Investment: $77.97bn
* Current Value: $81.79bn
* Portfolio Growth: $3.82bn
* Average Returns: 4.96%

---

## 📈 Dashboard Features

### 1. Portfolio Performance Summary

* Total clients, investment, returns
* Profit & loss tracking
* Portfolio growth trend

### 2. Client Segmentation

* By Age
* By City
* By Currency Type

### 3. Investment Analysis

* Asset Type (Equity, Mutual Funds, Insurance, etc.)
* Risk Level distribution
* Provider-wise investment

### 4. Time-Based Trends

* Returns over time
* Current value trend

### 5. Drill-down (Client Level)

* Individual client portfolio
* Investment breakdown
* Profit/loss tracking

---

## 🔍 Key Insights

* Majority of revenue comes from **top-performing clients**
* **Equity & Mutual Funds** dominate investment distribution
* Certain regions (e.g., Bangalore) show higher concentration of investors
* Portfolio value shows **declining trend in recent dates**
* High-risk assets contribute significantly to returns
